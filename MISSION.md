# Mission

**Pass the AD0-E136 — Adobe Experience Manager as a Cloud Service Migration Expert
certification.**

## Who
- Experienced **AEM developer** — components, templates, Sling, OSGi, HTL.
- Preparing for the **AD0-E136** expert-level certification exam.

## The exam (target)
- **50 questions · 100 minutes · pass = 31/50 (62%).**
- Multiple-choice / multiple-response, proctored.
- Official details: certification.adobe.com/courses/167

## Exam domains (weighting — drives lesson priority)
1. **Assessment — 32%** — BPA, migration plan/estimate, users/groups/permissions to move,
   deprecated capabilities, workflow requirements, content-transfer requirements,
   asset impact, dispatcher requirements.
2. **Design — 16%** — validate components vs latest, asset microservices config,
   caching strategy.
3. **Deployment — 16%** — Cloud Manager pipelines & environments, provision AEMaaCS
   envs (dev/stage/prod × author/publish/preview), Adobe I/O CLI.
4. **Migration — 20%** — execute tests, asset & base-content migration strategy,
   run migration tools (Content Transfer, Modernization, Refactoring), post go-live.
5. **Implementation — 16%** — merge projects into one pipeline-deployable project,
   RepoInit scripts for users/groups/permissions, local SDK, run-mode config.

## What success looks like
Answer exam-style questions correctly across all five domains; explain the *why*
behind each Adobe recommendation, not just recall facts.

## Framing for lessons
- **Exam-domain-mapped**: every lesson tags its domain + weight.
- **Prioritize by weight**: Assessment (32%) gets the deepest coverage.
- Exam-style multiple-choice practice (mirror the real question format).
- Ground in Adobe Experience League docs (primary source), never parametric memory.
